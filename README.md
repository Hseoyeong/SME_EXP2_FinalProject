## 1. 모티베이션 & 인트로
스마트모빌리티공학실험2를 진행하며, UWB/WiFi 신호, 환경의 영향, 위치 측위 계산 이론)에 대해 배우는 과정을 거치며 위치측위에 대한 기본 지식을 쌓는 시간을 가졌다. 초반의 수업은 직접 UWB/WiFi 신호를 활용해 거리를 수집하여 기본 위치 측위 방법인 삼변측량법을 적용해보고, 오차를 구하는 시간을 가졌다. 수업을 진행하면서 신호의 특성을 파악하여 위치 측위의 오차를 줄일 수 있는 방법들이 궁금해졌고, 이를 모두 활용하여 완성한 최종 프로젝트인 개인 위치측위알고리즘 개발 과제를 소개하려고 한다.
실생활 환경 및 현재 프로젝트에서 주어진 환경인 NLOS(Non-Line-Of-Sight)에서는 전처리 과정이 매우 중요하다.  장애물이 많은 환경에서 신호의 품질은 믿을 수 없기 때문이다. 다양한 방법으로 데이터 값을 보정하고, 이 값을 활용하여 궁금한 좌표를 최대한 정확히 계산해야하기 때문에, 좌표를 계산하는 과정도 매우 중요하다.
