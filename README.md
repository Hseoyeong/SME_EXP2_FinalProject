# SME_EXP2_FinalProject

## 1. 모티베이션 & 인트로
스마트모빌리티공학실험2를 진행하며, UWB/WiFi 신호, 환경의 영향, 위치 측위 계산 이론)에 대해 배우는 과정을 거치며 위치측위에 대한 기본 지식을 쌓는 시간을 가졌다. 초반의 수업은 직접 UWB/WiFi 신호를 활용해 거리를 수집하여 기본 위치 측위 방법인 삼변측량법을 적용해보고, 오차를 구하는 시간을 가졌다. 수업을 진행하면서 신호의 특성을 파악하여 위치 측위의 오차를 줄일 수 있는 방법들이 궁금해졌고, 이를 모두 활용하여 완성한 최종 프로젝트인 개인 위치측위알고리즘 개발 과제를 소개하려고 한다. 알고리즘의 전체적인 Pipeline은 (1) data set 전처리, (2) 머신러닝(Random Forest)을 기반한 데이터 품질 분석 (3) RANSAC(Random Sample Consensus) 기법+WLS,  HDOP 기법을 활용한 최종 좌표 도출 로 이루어진다.
실험 초반 과정과 최종 과정에서는 머신러닝 모델 선정의 변화가 있었다. 먼저 머신러닝을 도입하기로 한 이유는 실험 환경이 지닌 비선형적인 노이즈와 NLOS(Non-Line-of-Sight)특성을 단순히 수학적 계산만으로 모델링하는 데에 한계를 느꼈기 때문이다. 기존의 삼변측량법이나 최소제곱법(LS)는 모든 앵커의 신호 품질을 동일하게 취급하므로, 벽면 반사로 인해 발생한 불량 신호가 하나만 섞여도 전체 좌표가 왜곡된다. 이에 불량 신호가 발생하는 패턴을 학습하여 앵커 신호의 정밀도를 예측할 수 있는 머신러닝을 1차 필터링 시스템으로 도입하고자 했다. 
초반 실험 단계에서는 AdaBoost 모델을 선정하였다. 순차적으로 약한 분류기를 결합하여 이전 단계에서 잘못 분류된 데이터 샘플에 더 큰 가중치를 부여하는 부스팅 알고리즘이다. 초기 학습 데이터셋에서는 LOS/NLOS 분류 정확도가 매우 높게 도출되어 이상적인 모델로 판단되었다. 
그러나 실제 측위 알고리즘과 결합하여 최종 좌표를 도출하는 과정에서, data set에는 측정 환경에 의해 발생한 무작위성 Outlier가 다수 존재하였다. AdaBoost는 가중치를 업데이트하므로, 이러한 무작위성 노이즈와Outlier들을 반드시 학습하는 데에 집착하여 결과적으로 모델이 노이즈 자체를 학습해 버리는 과적합(Overfitting) 문제가 발생하여 최종 신호 품질 점수를 잘못 도출하여 전체 알고리즘의 성능을 저하시켰다.

따라서 최종 알고리즘에는 오차 가중치를 업데이트하는 Boosting 계열 대신 무작위로 분산 학습시켜 극단적인 Outlier에 흔들리지 않는 Bagging 계열을 사용하였다. Random Forest는 무작위로 추출된 샘플을 바탕으로 수많은 결정트리를 독립적으로 병렬 학습시킨다. 따라서 특정 트리가 노이즈와 Outlier에 과적합되더라도, 많은 다른 트리들이 도출한 결과에 의해 노이즈의 영향력이 상쇄된다. 또한 Random Forest는 수많은 트리의 결정을 통해 0~1 사이의 연속적인 확률을 안정적으로 출력한다.
 
## 2. 알고리즘 설명
개발한 위치 측위 알고리즘은 앵커 신호의 품질을 평가하는 머신러닝 학습 및 전처리(train.py)와 이를 활용해 최종 좌표를 도출하는 측위 엔진(main.py)로 구동된다. 아래는 먼저 앵커의 정밀도를 분류하고 머신러닝 모델 학습 과정에 대한 내용이다. 


1.1 데이터 로드 및 차원 분석
시스템은 마그네틱 데이터셋 파일로부터 고정 앵커의 좌표(BS_positions)와 각 노드별 앵커 측정 거리 행렬(d_hat), 그리고 검증을 위한 노드의 실제 좌표 행렬(p)을 호출한다. 고정 앵커의 좌표는 $2 \times 18$ 차원의 행렬로 구성되며, 각 노드별 앵커 측정 거리는 $18 \times 700$ 차원의 공간을 형성한다. 즉, 총 18개의 고정 앵커 기지국이 존재하며, 측위 대상이 되는 총 사용자 수는 700명으로 정의된다.

1.2 수학적 전처리 및 4차원 Feature 추출
수집된 데이터는 NLOS 환경이므로 장애물들로 인해 신호의 품질을 신뢰하기 어려우므로, 모든 노드와 모든 앵커의 조합을 돌며 다음과 같은 규칙으로 4차원 피처 공간($X$)을 구축한다.
(1) 물리적 한계 필터링: 측정된 거리 데이터가 $0\text{m}$ 이하이거나, 실내 공간의 이론적 최대 거리를 초과하는 $136\text{m}$ 이상일 경우, 오차로 간주하여 데이터셋 생성 대상에서 제외한다. 
(2) 기하학적 고유 피처 정의: 필터링을 통과한 데이터에 대해 머신러닝 모델의 필터링을 돕기 위해 총 4개의 기하학적 성분을 하나의 샘플 벡터로 병합한다. 
첫 번째 성분: 1차 필터링을 통과한 유효 측정 거리 값
두 번째 성분: 해당 신호를 송출한 고정 앵커 기지국의 X축 좌표
세 번째 성분: 해당 신호를 송출한 고정 앵커 기지국의 Y축 좌표
네 번째 성분: 측정 거리 값에서 정수 성분을 차감하여 도출한 소수점 이하의 고유 패턴 값
$$\text{Decimal Part} = \text{Distance} - \lfloor\text{Distance}\rfloor$$

1.3 오차 기반의 Labeling
머신러닝 모델에게 좋은 신호(LOS)와 나쁜 신호(NLOS)의 기준을 학습시키기 위해, p 값을 활용하여 가이드라인($y$)을 형성한다. 먼저, $(BS_x, BS_y)$와 $(p_x, p_y)$ 사이의 참 거리($d_{\text{true}}$)를 아래 유클리디안 기하학 공식으로 도출한다. $$d_{\text{true}} = \sqrt{(BS_x - p_x)^2 + (BS_y - p_y)^2}$$이후 실제 측정된 거리와 참 거리 사이의 절대 오차($\text{Error}$)를 계산한다.
$$\text{Error} = | \text{Distance} - d_{\text{true}} |$$
최적화 시뮬레이션을 통해 도출한 임계값인 오차 $2.5\text{m}$를 기준으 라벨링을 수행한다. 절대 오차가 $2.5\text{m}$ 이하일 경우 정상적인 직선 경로 신호(LOS)로 판단하여 라벨 1을 부여하고, 오차가 $2.5\text{m}$를 초과하여 치명적인 왜곡이 발생한 신호는 가짜 신호(NLOS)로 판단하여 라벨 0으로 이진화한다.

1.4 데이터 분할 및 Random Forest 파라미터 설계
구축된 전체 4차원 피처 행렬과 이진 라벨 배열을 머신러닝 모델의 과적합 방지 및 성능 확인을 위해 80%의 학습 세트(Train Set)와 20%의 검증 세트(Test Set)로 수학적 무작위 분할을 수행한다.
그 다음 아래와 같이 Random Forest Classifier의 파라미터를 설계하여 진행한다.

의사결정 트리 개수 (n_estimators): 총 100개의 독립적인 결정 트리를 병렬 배치하여, 특정 이상치 데이터가 전체 확률 분화에 영향을 주지 않도록 한다.
최대 깊이 제한 (max_depth): 과적합을 막고 실전 환경에서의 성능을 극대화하기 위해 각 트리의 최대 성장 깊이를 8로 제한한다.
클래스 가중치 보정 (class_weight): 데이터셋 내에서 정상 신호(LOS)와 불량 신호(NLOS)의 비율이 불균형하게 분포할 경우 모델이 한쪽으로 편향되는 것을 막기 위해, 가중치 밸런싱(balanced) 옵션을 적용하여 마이너 클래스에 대한 변별력을 인위적으로 높인다.

1.5 데이터 학습 및 최적 모델 저
직결 학습 전략을 취하여 처음부터 수집된 모든 데이터를 입력하여 Random Forest 모델을 즉시 학습시킨다.
최종 학습이 완료된 가중치 모델은 adaboost_model.pkl으로 로컬 시스템에 안전하게 내보내기(Export)가 되어, 다음 단계인 main.py 측위 엔진이 RANSAC을 수행할 때 확률 가중치 필터로 영구 참조된다.


서론의 Pipeline을 바탕으로, 위치 측위 엔진(main.py)은 머신러닝이 예측한 신호의 품질과 통계적 예측력과 유클리디안 기하학 지표를 무작위 가설 검증 프레임워크 내에 유기적으로 통합하였다. 본 시스템의 수학적 모델과 5단계 파이프라인의 구체적인 동작 원리는 다음과 같다.

2.1 수집 데이터의 1차 물리 필터링 및 차원 분석
매 차례 호출되는 사용자별 측정 거리 벡터($d\_hat\_u$)는 환경의 영향으로 다수의 노이즈를 포함하므로 수학적으로 검증을 거친다. 측정된 거리 값이 $0\text{m}$ 이하이거나 측정 공간의 이론적 대각선 한계 거리인 $136\text{m}$를 초과하는 데이터들을 이상치(Outlier)로 판정하여 연산에서 제외한다. 유효한 데이터가 최소 3개 미만일 경우에는 측위가 불가능하므로 예외적으로 원점 좌표를 반환하도록 한다.

2.2 머신러닝 기반 신호 품질 예측 및 유효 앵커 선별
1차 필터링을 통과한 앵커들에 대하여 에 학습 완료된 Random Forest 가중치 모델을 적용한다. 각 유효 앵커의 측정 거리, 고정 앵커의 X축 및 Y축 좌표, 그리고 측정 거리의 소수점 이하 성분($\text{Distance} - \lfloor\text{Distance}\rfloor$)으로 4차원 입력 피처 행렬을 구성한다. 모델은 이 행렬을 기반으로 각 앵커 신호가 정상 직선 경로(LOS)일 확률에 대응하는 품질 점수($w$)를 0~1 사이의 실수로 출력한다. 18개의 전체 앵커 정보를 흡수할 경우 걸러지지 못한 NLOS 오차가 전체 연산에 해를 끼치므로, 도출된 품질 점수를 내림차순 정렬하여 신뢰도가 높은 상위 7개의 앵커($k\_best = 7$)만을 RANSAC 후보군으로 선택하고 나머지는 배제한다.

2.3 가중 확률 분포 기반 RANSAC 및 가설 생성
선별된 7개의 앵커를 대상으로 총 80회($max\_iter = 80$)의 무작위 가설 검증 루프를 작동시킨다. 80회 실행으로 상위 7개 중 평면 측위의 최소 조건인 3개의 앵커 조합을 선택하는 수학적 조합 수인 $\binom{7}{3} = 35$가지를 완전히 포함하므로 이론상 모든 가설 공간의 완전 탐색이 가능하다. 이때, 무작위 선택은 단순 균등 분포가 아닌, 위에서 구한 머신러닝 품질 점수의 합이 $1$이 되도록 정규화한 가중 확률 분포(P)를 따른다. $$P_i = \frac{w_i}{\sum_{j=1}^{7} w_j}$$
이 확률 가중치 분포를 통해 신호의 품질이 우수한 앵커 조합이 샘플로 선정될 확률을 높인다. 루프마다 선택된 3개의 샘플 앵커 좌표와 측정 거리, 품질 점수를 독립 연산 엔진으로 전달한다.

2.4 가중 최소제곱법(WLS) 기반 좌표 최적화
선택된 3개의 샘플 앵커에 대하여 머신러닝 가중치를 반영한 비선형 가중 최소제곱법(WLS)을 정의한다. 구하고자 하는 임시 노드 가설 좌표를 $\mathbf{x} = [x, y]^T$라 할 때, 잔차 함수 벡터의 각 성분 $r_i$는 다음과 같이 정의된다.
$$r_i(\mathbf{x}) = w_i \cdot \left( \sqrt{(x - BS_{x,i})^2 + (y - BS_{y,i})^2} - \text{Distance}_i \right) \quad (i = 1, 2, 3)$$
초기 예측 지점을 3개 샘플 앵커의 무게중심으로 설정한 후, 비선형 최소제곱법 최적화 함수를 통해 잔차의 제곱합이 최소가 되는 임시 추정 좌표($est\_pos$)를 연산한다. 이때 실행 시간을 줄이기 위해 최적화 함수 내부의 공간 제약 범위 옵션을 제거하여 선형 탐색(Levenberg-Marquardt 변형 알고리즘) 구조로 구동되도록 한다.대신, 범위 제한 해제로 인해 좌표가 영역 밖으로 이탈하는 상황을 방지하기 위해 연산이 완료된 직후 후처리 클리핑(Clipping) 수학 함수를 적용하여 X축과 Y축 좌표를 각각 물리적 한계선인 $[-70.0, 70.0]$ 경계 내부로 강제 정사영 시킨다.

2.5 HDOP 기반 기하학적 정밀도 연산 및 점수 평가
추정된 가설 좌표를 추가 검증하기 위해 HDOP 연산 및 하이브리드 평가 함수를 실행한다. 먼저, 추정된 가설 좌표 $\mathbf{x} = [x, y]^T$와 가설을 세우는 데 사용된 3개의 샘플 앵커 간의 기하학적 편미분 성분으로 구성된 야코비안 설계 행렬 $A$를 설계한다.
$$A = \begin{bmatrix} 
\frac{x - BS_{x,1}}{\rho_1} & \frac{y - BS_{y,1}}{\rho_1} \\ 
\frac{x - BS_{x,2}}{\rho_2} & \frac{y - BS_{y,2}}{\rho_2} \\ 
\frac{x - BS_{x,3}}{\rho_3} & \frac{y - BS_{y,3}}{\rho_3} 
\end{bmatrix} \quad \left(\text{단, } \rho_i = \sqrt{(x - BS_{x,i})^2 + (y - BS_{y,i})^2}\right)$$
이 행렬을 기반으로 오차 공분산의 기준이 되는 공액 행렬 $H$를 산출하고, X축과 Y축의 대각 성분의 합에 제곱근을 취하여 최종 HDOP(기하학적 정밀도 저하율) 지표를 도출한다.
$$H = (A^T A)^{-1}$$$$\text{HDOP} = \sqrt{H_{1,1} + H_{2,2}}$$
그 후, 가설 좌표가 RANSAC 후보군인 상위 7개 앵커들과 비교하여 물리적으로 일치하는 정도를 나타내는 가중 절대 잔차 오차의 합($\text{Cost}$)을 계산한다.
$$\text{Cost} = \sum_{j=1}^{7} w_j \cdot | \rho_j - \text{Distance}_j |$$
최종적으로 단순 오차 합이 지닌 왜곡 가능성을 차단하기 위해, 오차 합에 HDOP 성분을 비선형 벌점 항으로 결합한 Final Score 함수를 통해 매 루프의 가설 좌표를 평가한다.
$$\text{Safe HDOP} = \min(\text{HDOP}, 10.0)$$
$$\text{Final Score} = \text{Cost} \times (\text{Safe HDOP})^{1.5}$$
가설 좌표의 거리 오차가 아무리 작더라도 앵커 배치가 일렬로 쏠려 기하학적 신뢰도가 낮으면 벌점이 폭발적으로 증가하여 정답 후보에서 탈락한다. 80회의 전체 루프 동안 이 하이브리드 스코어가 가장 최소화된 최적의 가설 좌표를 사용자의 최종 2차원 위치 정답으로 선정되어 반환하며 알고리즘이 끝난다.
